# PortfolioPilot AI 🚀
A rule-based intelligent portfolio analysis assistant built as an MS Finance student project at Simon Business School, University of Rochester.

## Live Demo
👉 https://yourusername.github.io/portfoliopilot-ai/

## Features
- AI Portfolio Score (0–100) with 4 sub-scores
- Risk appetite input (Conservative → Speculative)
- Buy / Hold / Sell signals per position
- 32+ personalized stock recommendations
- Concentration alerts, sector charts, rebalancing plan

## Tech Stack
HTML · CSS · JavaScript · Chart.js · Rule-Based Logic

## Disclaimer
Educational use only. Not financial advice.
